# Reledger Website

Landing page for [Reledger](https://github.com/faisal-hendra/reledger), a personal ledger and financial journaling desktop application.

## About

Reledger is a 100% local, lightweight desktop app for tracking personal finances. Built with Electron, React, and TypeScript.

## Tech Stack

- **Astro** — Static site generator
- **TypeScript** — Type-safe JavaScript
- **OKLCH** — Modern color space for theming

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

src/
